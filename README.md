# Clonando a home page do IPCA

Caros alunos este repositório servirá de apoio a componente pratica e teórica da UC Programação Javascript, especificamente para o módulo Fundamentos Web.

## Tarefa:
* Clonar repositório (Utilize o terminal do VSCode);
* Crie uma nova branch com o nome de: change-page-title;
* Altere o elemento ``` <title></title> ``` inserindo o texto: IPCA - Ensino Superior Público - Barcelos;
* Commite a alteração realizada (Utilize o Commit Semântico);
* De um Push;
* Volte a página do github e crie um pull request;
* Commite as mudanças;
* Volte ao VSCode, altere para a branch main;
* Na branch main faça um pull para buscar as atualizações;
* Siga trabalhando sempre criando uma nova branch para novas atualizações do projeto.
